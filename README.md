# A React Redux (State Management) webapp project.
This app functions as an e-Commerce where users can:
* Check Sales Overview
* Can add item to Cart
* Can access Cart

Developed using React Redux and D3 library (Victory Chart)

Testing:
* Unit tests
* Cypress e2e tests
